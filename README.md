# Jarilo

**Jarylo** (Cyrillic: Ярило or Ярила; Polish: Jaryło; Croatian: Jura or Juraj; Serbian: Jarilo; Slavic: Jarovit), alternatively Yarylo, Iarilo, or Gerovit, is a Slavic god of vegetation, fertility and springtime.
